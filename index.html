CURRENT ONGOING PROJECT

This UBC query explorer I am working on is a simple database that provides a list of either UBC courses or rooms given a query with a matching criteria. Specifically, it allows users to add data sets of UBC courses, and then proceeds to make some query like show me all courses with an average > 60% or show me all courses that are taught by this instructor. The project consists mostly of TypeScript for the backend while the requests and results are in JSON. The majority of the code involves asynchronous code, using promises. Tests are written with Mocha and Chai.

This project can be found on github .

FLOOD FILL

This is a program that executes a "flood fill" on a PNG image when specified a point on the image and a fill pattern. The code uses a breadth-first-search and depth-first-search to create fill algorithms to fill images with rainbow solid, gradient, and grid fill patterns.

EARTHQUAKE EFFECTS

A website containing Kepler.GL visualizations of major international earthquakes based on deaths and injuries caused, and the magnitude and depth of the earthquake. Every earthquake contains one of the following characteristics: approximately $1 million or more damage, 10+ deaths, magnitudes of at least 7.5, modified Mercalli intensity X or greater, or a tsunami generated.
